Windows XP React
================

Esta aplicacao recria uma experiencia inspirada no Windows XP usando React e Tailwind CSS.

Recursos principais:
- tela de boot
- login com usuario e senha
- tela de boas-vindas
- area de trabalho com barra de tarefas
- janelas arrastaveis
- menu iniciar
- pasta "Meus Documentos" com arquivos reais
- Bloco de Notas capaz de abrir arquivos .txt
